## 📁 Portfolio Table of Contents

### 1. 🎬 IMDB Movie Trend & Studio Strategy Pipeline (`/01_imdb_movie_analysis`)
* **Core Problem Solved:** Raw, messy entertainment metrics (such as string runtimes, currency strings, and multi-actor arrays) were blocking downstream analysis. Furthermore, the studio lacked a data-backed blueprint for investing production capital.
* **The Engineering Solution:** Built an in-place parsing pipeline to normalize financial and duration metrics. Handled multi-director and cast arrays using index explosions to maintain accurate revenue reporting. 
* **The Analytical Outcome:** Discovered a massive industry divergence - while overall movie ratings have declined historically, sequel revenues have surged. Generated a three-step corporate roadmap to optimize studio capital by prioritizing franchise assets and shifting marketing focus to audience social volume.
